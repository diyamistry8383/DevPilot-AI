## Overview

DevPilot AI is an AI-powered engineering productivity platform that helps software development teams manage projects, analyze engineering productivity, automate sprint planning, generate intelligent reports, and provide AI-driven recommendations.

---

## Core Features

- AI Assistant
- Project & Sprint Management
- Zoho Sprints Integration
- Multi-Agent AI
- RAG Knowledge Hub
- AI Resume Generator
- AI Learning Recommendation
- Engineering Analytics Dashboard
- Predictive Analytics
- Smart Automation
